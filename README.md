# 新中特选择题刷题

这是一个纯静态刷题网页，可直接通过 GitHub Pages 发布，手机浏览器可访问。

## 本地打开

直接打开 `index.html` 即可使用。

## GitHub Pages 发布

1. 在 GitHub 新建一个仓库，例如 `xinzhongte-xztk`。
2. 将本目录推送到该仓库。
3. 在仓库 `Settings -> Pages` 中选择 `Deploy from a branch`，分支选 `main`，目录选 `/root`。
4. 保存后等待 GitHub Pages 生成访问地址。

题库数据在 `data.js` 和 `questions.json` 中；网页入口是 `index.html`。
